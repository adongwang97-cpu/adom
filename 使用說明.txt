阿東寫真站 — 使用說明
==============================
網站：https://adom.adongwang97.workers.dev/
後台：https://adom.adongwang97.workers.dev/admin/

【後台管理(推薦，會用瀏覽器點滑鼠就會用)】
去 /admin/ 輸入密碼登入(密碼是你在 CF 設的 ADMIN_PASS，見下方「後台設定」)，可以：
- 每個相簿分類：編輯故事文案、刪除/排序照片
- 新增/刪除/排序整個相簿分類
- 主題風格：目前只有「玫瑰金 Glam」一套，要加新風格跟我說一聲

※ 這個後台不做「網頁上傳圖片」。新照片一樣是 git push 進 photos/ 資料夾(我幫你轉webp+分類)，
   推上去後照片會自動出現在對應分類，然後就可以在後台排序/寫文案/刪除。

【後台設定(第一次用要先在 CF 儀表板點幾下)】
去 Cloudflare 儀表板 → Workers & Pages → adom → Settings：
1. D1：先在左側「Storage & Databases → D1 SQL Database」建一個資料庫(名字隨意，例如 adom-db)
   → 回 adom → Settings → Bindings → Add binding → 選 D1 Database
   → Variable name 填「DB」(一定要這個名字，大寫) → 選剛建的資料庫
2. Settings → Variables and Secrets → Add：
   - ADMIN_PASS = 你要設的後台密碼
   - ADMIN_SECRET = 任意一長串亂碼(簽章用，不用記得住)
3. 存檔，重新整理 /admin/ 即可登入

表格(sections/photo_state/settings/donations)會在第一次真的呼叫 API 時自動建立，
不用手動去 D1 主控台貼 SQL。

沒做這幾步之前，網站本身完全正常(照舊顯示現有照片)，只是 /admin/ 的編輯功能會提示「尚未綁定」。

【抖內三階(USDT，OxaPay 金流)】
src/worker.js 最上面 DONATE_TIERS 常數可改金額/文案，目前：
  應援 Bronze 5 USDT｜VIP Silver 15 USDT｜摯友 Gold 30 USDT
index.html 裡也有一份一樣的 DONATE_TIERS(前端顯示用)，改的話兩邊要一起改。

要讓「線上刷卡」真的能用，需要：
1. 去 OxaPay 官網開一個「阿東專屬」商戶帳號(⚠️不要跟 picks168 共用同一組 key，
   金流帳號共用等於鏈上/後台把阿東跟 ys89/預測站兜在一起，失去分開經營的意義)
2. 拿到 merchant API key → CF adom → Settings → Variables and Secrets → Add
   → OXAPAY_API_KEY = 你的 key
3. 存檔即可，網站的「選一個等級開始」按鈕會自動改用真金流(靠 D1 存訂單狀態)

沒設定 OXAPAY_API_KEY 之前，抖內彈窗會顯示「線上刷卡尚未開通」，
但下面「直接轉帳(彈性金額)」那組 QR/地址仍正常可用。

抖內成功後會自動：
- 帶回網站顯示感謝頁 + 對應福利(解鎖高清寫真 / 進私密頻道連結)
- 如果有填暱稱，出現在首頁「感謝牆」(資料來自 D1 donations 表，已付款才會顯示)

【私密頻道 — 單次專屬邀請連結(防連結被轉傳濫用)】
Silver/Gold 抖內確認付款後，系統會用 bot(@Adombaby520_bot)現生一條「只能用一次、48小時內有效」
的專屬邀請連結給該粉絲，不是固定連結，別人轉傳也進不去。設定：
1. 把 @Adombaby520_bot 加成私密頻道的「管理員」，至少要給「邀請使用者連結」權限
2. CF adom → Settings → Variables and Secrets → Add → TG_BOT_TOKEN = bot 的 API token
   (BotFather 給的那串，⚠️ 這也是憑證，別再貼進對話，直接貼 CF 表單就好)
3. src/worker.js 最上面 TG_CHANNEL_ID 常數要填頻道的數字 ID(不是邀請連結)，
   跟 Claude 說「頻道管理員加好了」，Claude 會幫你抓出這個 ID 並填上去
4. 沒設定好之前，系統會自動退回用固定的 VIP_CHANNEL_URL(不會開天窗)

【頻道自動發文(排程輪播)】
去 /admin/ 「📢 頻道自動發文」區塊：
- 填文案(可選一張現有照片配圖) → 加入排程
- 系統每天台灣晚上 8 點自動挑「最久沒發過」的一則發到私密頻道，天然輪播，
  新增/刪除都自動處理，不用手動排順序
- 「🚀 立刻試發一則」按鈕可以馬上測試設定對不對，不用等到晚上8點
- 要改發文頻率：wrangler.toml 裡 `crons = ["0 12 * * *"]` 這行改一改(找 Claude 改最快)

跟私密頻道單次連結共用同一組 TG_BOT_TOKEN + TG_CHANNEL_ID，設定好一次兩個功能都會通。

【解鎖碼 / 私密頻道固定連結 / 收款地址】
打開 index.html 最上面：
   var WALLET='TXXXX...';           ← 阿東的 USDT 收款地址(彈性抖內QR用)
   var UNLOCK_CODE='adong520';      ← 手動解鎖碼(私訊發，或抖內Bronze以上自動解鎖)
   var VIP_CHANNEL_URL='https://t.me/+-h2PrPh-_0w3MjY1';  ← 頻道固定連結(解鎖碼流程用+單次連結的備援)
   var AGENT_URL='...';             ← 夜色代理連結(已接代理碼 rs82gq)
   var PREDICT_URL='...';           ← 預測站廣告LP(已接 KOL=adong 追蹤)

★ WALLET 務必用「全新、獨立」的 USDT 錢包，OXAPAY_API_KEY 務必用「阿東專屬」商戶帳號，
  不要跟預測站/ys89 共用 — 否則鏈上/後台紀錄會把兩邊兜在一起，失去分開經營的意義。

【KOL 帶量怎麼看】
阿東帶去預測站的人，在 picks168 後台「🎯 KOL 代碼」分頁看得到成效(代碼 adong)。
夜色帶量另外看 ys89 代理後台的代理碼 rs82gq。

【CF 上目前有 2 個部署，adom-2ar.pages.dev 那個是舊的、建議刪掉】
加 wrangler.toml 後 CF 自動多拆出一個舊版靜態 Pages 專案(adom-2ar.pages.dev)，
跟真正在用的 Worker 版(adom.adongwang97.workers.dev)是分開的兩份，容易搞混，
確認新版都正常後可以去 CF 儀表板把 adom-2ar 那個刪掉，只留 Worker 版。

【部署】
git push 到 GitHub(adongwang97-cpu/adom)即可，CF 有連 Git 會自動重新部署。
⚠️ 不要再用 GitHub 網頁拖拉上傳檔案 — 之前這樣做過一次，把新版蓋回舊版，
  導致要用 force push 才修復。新檔案都交給 Claude 在本機改好 commit，
  你只要負責在終端機跑 git push 這一步。
